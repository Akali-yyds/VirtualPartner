# VirtualPartner 未来开发候选清单

更新时间：2026-05-19

本文档记录已经讨论过、但当前没有纳入完成态的候选方向。它不是当前 TODO，也不表示这些能力已经实现。

进入任何候选方向前，仍需先按 [ReadFirst.md](./ReadFirst.md) 重新讨论目标、边界、验收方式和实现计划。

## App Shell 与桌面应用体验

- Settings Panel：应用内设置面板。
- 设置保存与重置：保存用户偏好并支持恢复默认。
- 窗口 / 全屏 / 分辨率切换：提供常用显示模式选择。
- `F11` 全屏切换。
- Master / TTS / UI 音量控制。
- 启动状态检查页：展示 LLM、TTS、ASR、本地资源状态。
- 日志查看器：面向用户或开发者的运行日志入口。
- 最小化到托盘、后台运行策略、低功耗模式。
- 透明桌宠窗口、窗口置顶、无边框拖拽桌面宠物模式。
- About / 帮助页面。
- Release / Debug 双模式打包流程。

## 角色与交互

- 多角色同时在场调度。
- 多角色之间主动对话。
- Momotalk 群聊或多角色聊天。
- 更完整的多角色切换 UI。
- 道具 / 家具交互系统。
- 场景物体交互。
- 更复杂的情绪人格系统。
- 精细 LookAt、眼睛注视和眨眼策略。
- 手指动作库或手势系统。

## 行为与移动

- NavMesh 路径规划。
- 复杂避障。
- 自动回退、重新规划和多段路径。
- 更自然的 idle 行为组合。
- 更丰富的 locomotion 动画和转身衔接。

## 语音与表情

- 更完整的 Toki 专属音色训练与管理。
- 多角色 voiceId 管理界面。
- 音素级口型。
- 更丰富的 ExpressionProfile 和面部骨骼姿态。
- 表情与情绪状态的长期关联。
- 实时流式字幕。
- 唤醒词。
- 说话人识别。

## 记忆与智能

- RAG 记忆库。
- 向量数据库。
- 关键词检索与摘要压缩。
- 记忆管理 UI。
- 多角色共享或隔离记忆策略。
- 更严格的敏感信息处理策略。

## 平台与发布

- Android 平台适配。
- 自动化构建和版本发布流程。
- 首次启动引导。
- 用户数据备份与迁移。
- 本地服务一键启动和健康修复工具。
